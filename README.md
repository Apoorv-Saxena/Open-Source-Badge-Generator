# Open-Source-Badge-Generator

<b>Open Source Badge Generator</b> is a tool written in JS, that lets you create and download your own <i>Open Source Contributor</i> badge!

Here's how you can use it:
1. Head on to https://tharvid.github.io/Open-Source-Badge-Generator/
2. Enter your GitHub username in the text field and hit <b>GENERATE BADGE</b>.
3. This will generate a customized <i>Open Source Contributor</i> badge provided a valid GitHub username is entered.
4. Click on the <b>Click to download</b> button to download your badge.
5. Congratuations! You're done.
